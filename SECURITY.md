# Seguridad de Agender

## Versiones compatibles

Se mantiene la versión más reciente publicada de Agender. Las correcciones de
seguridad se distribuyen mediante una nueva versión firmada.

## Reportar una vulnerabilidad

No publiques vulnerabilidades, credenciales, claves privadas ni datos reales en
un issue público. Utiliza **Security > Report a vulnerability** en este
repositorio para enviar un reporte privado con:

- versión afectada y entorno;
- pasos mínimos para reproducir el problema;
- impacto observado o esperado;
- cualquier mitigación temporal conocida.

Se confirmará la recepción del reporte antes de divulgar información técnica.
